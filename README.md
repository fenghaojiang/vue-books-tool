# vue-books-tool  
在爬取豆瓣Top250书籍信息到数据库之后,在做这个前端的展示分析工具  

爬取豆瓣Top250的Go语言写的爬虫项目[douban-books-crawler](https://github.com/fenghaojiang/douban-books-crawler)  

  
# Run
**Backend**
运行后端代码

```shell
$ go run main.go
```

访问localhost的8000端口：
```shell
http://localhost:8000/douban
```  

**Frontend**
前端
```shell
$ cd ./vue-tool
```

```shell
$ npm run serve
```
